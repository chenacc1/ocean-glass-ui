# OceanAvenu Dark Glass UI

基于照片底图的暗色毛玻璃 UI 设计系统，适配 React + Ant Design 6。

![截图1](../../screenshots/yolo-1.png)

![截图2](../../screenshots/yolo-2.png)

## 快速使用

在 Claude Code 中加载此 skill 后，说 `ocean-glass-ui` 即可将这套风格应用到当前项目。

## 设计令牌速查

| 令牌 | 值 | 用途 |
|------|-----|------|
| 基底色 | `#07080a` | 页面底色 |
| 玻璃面板 | `rgba(7,8,10,0.12)` + `blur(40px)` | 卡片、侧栏、抽屉 |
| 输入框 | `rgba(7,8,10,0.08)` | 输入框背景 |
| 强调色 | `#78bbff` | 按钮、链接、聚焦 |
| 文字主色 | `rgba(255,255,255,0.92)` | 标题、正文 |
| 文字次色 | `rgba(255,255,255,0.6)` | 标签、提示 |
| 圆角面板 | `20px` | 卡片、弹窗 |
| 圆角输入 | `12px` | 按钮、输入框 |
| 阴影 | `0 4px 20px rgba(0,0,0,0.3)` | 所有浮层 |
